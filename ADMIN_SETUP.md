# 🏄 Time to Surf — Инструкция по настройке Админки

## Шаг 1: Создать таблицы в Supabase

1. Открой [Supabase Dashboard](https://supabase.com/dashboard) → твой проект
2. Перейди в **SQL Editor**
3. Вставь и выполни весь файл `supabase_schema_v2.sql`

Это создаст таблицы:
- `site_settings` — цены, места, метки расписания
- `camp_sessions` — смены лагеря
- `gallery_photos` — фотографии

## Что умеет админка:

### 💬 Отзывы
- Просматривать новые отзывы
- Публиковать / удалять

### 📅 Смены
- Добавить новую смену (даты, название на 3 языках, руководители, цвет, тип)
- Редактировать существующие
- Удалять смены
- Помечать «горящие» смены 🔥

### 📸 Фото
- Добавить фото по URL или пути `/имя.jpg`
- Выбрать раздел: Главные, На воде, Команда, Моменты
- Удалять фото

### 💰 Цены
- Изменить цену за 3 дня
- Изменить цену за 5 дней

### ⚙️ Настройки
- Количество мест занято / всего
- Дата ближайшей смены (текст типа "15.06")
- Метка года/сезона ("Лето 2026" → "Лето 2027")

## Как настройки из Supabase попадают на сайт

После сохранения в админке данные хранятся в Supabase.

Чтобы сайт автоматически подтягивал настройки — добавь в `page.tsx` в useEffect при загрузке:

```typescript
useEffect(() => {
  fetch('/api/settings')
    .then(r => r.json())
    .then(settings => {
      // Применить: spots_taken, spots_total, next_session_date, etc.
    })
}, [])
```
